# Ecommerce Sales EDA

## Project Overview

This project performs Exploratory Data Analysis (EDA) on an E-commerce Sales dataset.

## Objectives

* Calculate basic statistics
* Identify trends and patterns
* Detect outliers
* Generate visualizations
* Summarize key business insights

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Google Colab

## Dataset

The dataset contains information about customer orders, products, payment methods, referral sources, and sales.

## Analysis Performed

* Data Cleaning
* Missing Value Analysis
* Duplicate Check
* Descriptive Statistics
* Product Analysis
* Payment Method Analysis
* Outlier Detection
* Data Visualization

## Conclusion

The dataset was analyzed to identify sales patterns, customer behavior, and key business insights through Exploratory Data Analysis.
